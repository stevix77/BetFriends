# BetFriend : jeu de paris entre amis
Manage bets between friends

BetFriend : jeu de paris entre amis  
	Front : .NetMAUI, Angular, VueJS  
	Back : C# .Net, typescript NestJS, (PHP symfony)  
	
	- ajouter un ami
	- créer un pari
	- accepter/décliner un pari
	- donner le résultat d'un pari
	- lister les paris
	- créer un compte
	- authentification
